# Как устроен анимированный терминал в профиле

Профильный README — это одна анимированная SVG-«запись» терминальной сессии:
`ls` → `cd andromaquehere && ls` → `cat README.md` (ASCII-баннер) →
`cat skills.md` (таблица скиллов) → `cat projects.md` (таблица проектов) →
мигающий курсор. Никакого JS и внешних запросов — только SVG + CSS-анимации,
поэтому GitHub спокойно рендерит её через `<img>`.

## Файлы

| Файл | Что это |
|---|---|
| [gen_terminal.py](gen_terminal.py) | Генератор. Единственный источник правды — SVG руками не правим. |
| `../assets/terminal-dark.svg` / `terminal-light.svg` | Результат: тёмная и светлая версии, подключены в README через `<picture prefers-color-scheme>`. |
| [badges/](badges) / [badges-black/](badges-black) | Вендоренные бейджи shields.io. Из них извлекаются base64-логотипы (белые — для тёмной темы, чёрные — для светлой). |

## Как внести правку

```bash
pip install pyfiglet                 # один раз (или venv)
python scripts/gen_terminal.py      # перегенерить оба SVG
# бампнуть кэш-версию в README.md:
V=$(git hash-object assets/terminal-dark.svg | cut -c1-7)
sed -i '' "s|terminal-dark\.svg?v=[a-z0-9]*|terminal-dark.svg?v=$V|; s|terminal-light\.svg?v=[a-z0-9]*|terminal-light.svg?v=$V|" README.md
git add README.md assets && git commit && git push
```

Что где меняется в `gen_terminal.py`:

- **Сценарий** — список `SEQ`: `(промпт, команда, строки вывода, тип, отступы)`.
  Типы вывода: `instant` (появляется сразу, как настоящий `ls`), `letters`
  (баннер по буквам), `flush` (таблица построчно сверху вниз).
- **Скиллы** — `SKILL_COLS`: колонки `(заголовок, [(ключ-иконки, название), …])`.
  Ключ — имя файла в `badges/`; `None` = без иконки.
- **Проекты** — блок `projects` (строки таблицы, описание оборачивается `textwrap`).
- **Тайминги** — `CPS` (сек/символ печати) и паузы в блоке `timeline`.
- **Баннер** — `pyfiglet`, шрифт `ansi_shadow`, 120 колонок.

## Ключевые механики (и грабли)

- **Кэш GitHub.** Картинки README проксируются и кэшируются (camo/raw CDN).
  Без бампа `?v=` в URL пользователи видят старую версию вечно. Плюс кэш
  браузера — проверять через Cmd+Shift+R.
- **Выравнивание моноширинного арта.** Каждая строка — `<text textLength=…
  lengthAdjust="spacingAndGlyphs">`: прибивает ширину к сетке независимо от
  того, какой моноширинный шрифт у зрителя. Ширина символа `CW = 8.4px` при
  `font-size 14`.
- **«Печать»** — `clip-path: inset(0 100% 0 0) → 0` со `steps(N)`, N = число
  символов; курсор — `<rect>` с синхронным `translateX` тем же `steps()`.
  Построчный вывод таблиц — тот же приём по вертикали.
- **Иконки.** Внутри `<img>`-SVG нельзя грузить внешние URL — логотипы вшиты
  как `data:image/svg+xml;base64` (взяты из бейджей shields.io). У OpenAI и
  Codex иконок нет: simple-icons их удалил/не имеет — они текстом.
- **Ссылки внутри SVG не кликабельны** (контекст `<img>`). Всё, что должно
  быть ссылкой, — только текстом в таблице.
- **`<img>` в markdown**: каждый тег на своей строке исходника GitHub рендерит
  отдельным абзацем (вертикальный стек). Инлайновый ряд = один длинный ряд
  в одной строке исходника. Сейчас неактуально (картинка одна), но грабли
  задокументированы кровью.
- **Reduced motion** — `@media (prefers-reduced-motion: reduce)` показывает
  финальный статичный кадр.
- **Тёмная/светлая тема** — два отдельных SVG, переключение через
  `<picture><source media="(prefers-color-scheme: dark)">`. Это настройка ОС,
  не тема GitHub, но на практике совпадает.
